**🚛 V.E.L.O.S — Smart Logistics & Driver Assist System**

Real-time decision-support platform for logistics that combines weather, traffic, and route intelligence to improve driver safety and operational efficiency.

---

**Live Demo**
https://velos-logistics-system.web.app

---

**Problem**

- 20–25% of road accidents are weather-related  
- Existing fleet systems focus on dispatchers, not drivers  
- No unified “safety score” for quick decision-making  

---

**Solution**

- Built a driver-centric system that:
  - Integrates real-time weather + traffic data  
  - Computes a **Comfort Score** for route safety  
  - Provides multi-route comparison (fastest vs safest)  
  - Suggests safe stops (hospitals, fuel, rest areas)  
  - Enables emergency alerts with live location  

---

**Key Features**

- Weather-aware routing with live alerts  
- Comfort Score (risk classification engine)  
- Real-time vehicle tracking (Firebase)  
- Dual dashboard:
  - Map view (global monitoring)  
  - Table view (status + delay reasons)  
- Emergency panic system  
- Document vault for logistics data  

---

**Tech Stack**

- Frontend: React.js, Tailwind CSS  
- Backend: Firebase (Realtime DB, Auth)  
- APIs: Google Maps, OpenWeather  

---

**Architecture**

- Driver App → route insights + alerts  
- Admin Dashboard → monitoring + analytics  
- Firebase → real-time sync  
- External APIs → maps + weather  

---

**What Makes This Project Different**

- Driver-first UX (not just dispatcher tools)  
- Decision-support system (not forced automation)  
- Unique **Comfort Score abstraction**  
- Integration of safety + logistics + real-time data  

---

**Limitations**

- Depends on external APIs  
- Prototype-scale system  
- ML-based prediction not yet integrated  

---

**Future Work**

- ML-based route risk prediction  
- Driver fatigue detection  
- Advanced analytics dashboard
- live to use in daily life
